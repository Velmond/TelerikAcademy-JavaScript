07. Write a program that finds the index of given element in a sorted array of integers by using the binary search algorithm (find it in Wikipedia).

Enter the array as a string with the elements separated by commas
(example: 79,-2,35,4,-22,8,42,23,42,1,11,8,62,0,-4,17,-35,2,-65,0,-83)
Any NaNs will be removed from the array before executing
(example: 79,-2,35,4,a,-22,dw,8,42,,23,42,w1,11,8,62,-0, ,-4,17,-35,2r,-65,0,-83)
The string will be split into an array when you press 'Execute!'
