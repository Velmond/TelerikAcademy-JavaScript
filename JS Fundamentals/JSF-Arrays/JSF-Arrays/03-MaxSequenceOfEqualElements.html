03. Write a script that finds the maximal sequence of equal elements in an array.
    Example: {2, 1, 1, 2, 3, 3, 2, 2, 2, 1} => {2, 2, 2}.

Enter the arrays as a single string (example: 1,1,2,3,7,7,7,7,4,6,2,5,5,5,3).
Any NaNs will be removed before executing (example: 1,1, ,3,2w,,7,4d,7,4,6,r3,2,5,5,3).
The string will be split into an array when you press 'Execute!'
