05. Sorting an array means to arrange its elements in increasing order. Write a script to sort an array. Use the "selection sort" algorithm: Find the smallest element, move it at the first position, find the smallest from the rest, move it at the second position, etc. Hint: Use a second array

Enter the numbers as a string with the entries separated by commas
(example: 79,-2,35,4,-22,8,42,23,42,1,11,8,62,0,-4,17,-35,2,-65,0,-83)
Any NaNs will be removed before sorting
(example: 79,-2,35,4,a,-22,dw,8,42,,23,42,w1,11,8,62,-0, ,-4,17,-35,2r,-65,0,-83)
The string will be split into an array when you press 'Execute!'
