06. Write a program that finds the most frequent number in an array.
    Example: {4, 1, 1, 4, 2, 3, 4, 4, 1, 2, 4, 9, 3} => 4 (5 times)

Enter the numbers as a string with the entries separated by commas
(example: 79,-2,35,4,-22,8,42,23,42,1,11,8,62,0,-4,17,-35,2,-65,0,-83)
Any NaNs will be removed before executing.
(example: 79,-2,35,4,a,-22,dw,8,42,,23,42,w1,11,8,62,-0, ,-4,17,-35,2r,-65,0,-83)
The string will be split into an array when you press 'Execute!'
